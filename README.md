# To-Do Web App
## Oasis Infobyte Web Development Internship — Task 3

---

## Objective
To build a fully functional To-Do web app that allows users to add,
complete, edit, and delete daily tasks with date and time tracking.

---

## Files in This Project

| File       | Purpose                                          |
|------------|--------------------------------------------------|
| index.html | Page structure, form, task lists, modal popup    |
| style.css  | All styling, layout, grid, and responsiveness    |
| script.js  | All logic — add, complete, edit, delete tasks    |
| README.md  | Basic Documentation of this project (To-Do APP)  |

---

## Tools Used
- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript (Vanilla — no frameworks)
- Google Fonts (Poppins)
- VS Code + Live Server
- GitHub + GitHub Pages

---

## Steps Performed
1. Created 3 separate files — HTML, CSS, JavaScript
2. Built input form to add new tasks
3. Created Pending Tasks list and Completed Tasks list
4. Wrote JavaScript functions:
   - addTask() — adds new task to the list
   - renderTasks() — draws all tasks on screen
   - toggleComplete() — moves task between lists
   - deleteTask() — removes a task with confirmation
   - openEdit() / saveEdit() — edit task via popup modal
   - clearAllCompleted() — removes all completed tasks
5. Added date and time when task is added and completed
6. Added summary badges showing Total, Pending, Completed counts
7. Added keyboard shortcuts (Enter to add, Escape to close modal)
8. Made page responsive for mobile screens

---

## Features
- Add new tasks with date and time stamp
- Pending Tasks and Completed Tasks in separate lists
- Mark task complete — moves to Completed list automatically
- Unmark task — moves back to Pending list
- Edit any task via a popup modal
- Delete any task with confirmation dialog
- Clear all completed tasks at once
- Summary badge bar showing task counts
- Enter key to add tasks, Escape to close modal
- Fully responsive on mobile screens

---

## Outcome
A complete To-Do web application was built using only HTML, CSS, and
JavaScript with all required features including separate pending and
completed task lists, edit modal, and date/time tracking.
